# Оценка сопровождения и ежемесячных расходов (OPEX)
## Pilates Space — web + admin + backend (без mobile в расчете infra)

**Валюта:** EUR/месяц  
**Стек:** Vercel + Railway + Cloudflare + Sentry

---

## 1) Инфраструктура и SaaS

### Базовый (MVP, низкая нагрузка)
- Vercel (web + admin): 20–80€
- Railway (api + worker + postgres + redis): 80–220€
- Cloudflare (DNS/WAF): 0–20€
- Sentry: 0–30€
- Email/SMS/push сервисы (по необходимости): 10–40€

**Итого базово:** **110–390€/мес**

### Рабочий (стабильный прод, средняя нагрузка)
- Vercel: 40–120€
- Railway: 180–450€
- Cloudflare: 20–200€ (в зависимости от плана WAF)
- Sentry: 20–100€
- Транзакционные сервисы (email/SMS): 20–80€

**Итого рабочий:** **280–950€/мес**

### С запасом (рост, вторая локация, выше требования к uptime)
**Итого:** **900–1,800€/мес**

---

## 2) Сопровождение разработки (часы)

## Минимальное сопровождение
- Мониторинг, мелкие фиксы, обновления зависимостей,
- мелкие правки интерфейса,
- поддержка инцидентов.

**20–35 ч/мес**

## Нормальный операционный режим
- Всё выше + небольшие доработки,
- улучшение админки/отчетов,
- регулярные релизы.

**40–70 ч/мес**

## Активное развитие
- Параллельно с поддержкой идут новые фичи.

**80–120 ч/мес**

## 4) Совокупная месячная стоимость (ориентир)

## Режим «поддержка без активной разработки»
- Инфра: 280–950€
- Работа: 20–35 ч

При 50€/ч: **1,280–2,700€/мес**

## Режим «рабочий прод + регулярные улучшения»
- Инфра: 280–950€
- Работа: 40–70 ч

При 50€/ч: **2,280–4,450€/мес**

## Режим «активное развитие»
- Инфра: 900–1,800€
- Работа: 80–120 ч

При 50€/ч: **4,900–7,800€/мес**
